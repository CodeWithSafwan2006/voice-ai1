# 🎙️ Voice AI Assistant

## 🚀 Overview

Voice AI Assistant is a web-based application that allows users to interact using voice commands. It converts speech to text, processes user input, and can respond using text-to-speech, creating a smart and interactive AI experience directly in the browser.

---

## ✨ Features

* 🎤 Speech-to-Text (voice input recognition)
* 🔊 Text-to-Speech (AI voice response)
* 🤖 Basic AI response system
* ⚡ Real-time interaction
* 📱 Responsive design (mobile & desktop)

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/CodeWithSafwan2006/Voice-AI-Assistant.git
```

2. Navigate to the project folder:

```
cd Voice-AI-Assistant
```

3. Run the project:

* Open `index.html` in your browser
* OR use Live Server
